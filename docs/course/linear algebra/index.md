# 线性代数I/II(B)
中国科学院大学 “线性代数(B)” 课程采用的教材为 **《代数学引论》（柯斯特利金）**与 **《基础代数》（席南华）**。“线性代数I”为大一上学期公共必修课，“线性代数II”为大一下学期公共必修课。

本课程虽然正式名称为“线性代数”，但其中包括了少量抽象代数的入门知识（如群环域），读者可以结合自身需求阅读。
